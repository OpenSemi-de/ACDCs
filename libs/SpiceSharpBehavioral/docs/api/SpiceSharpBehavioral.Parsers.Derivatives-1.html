Declaration
public static double FudgeFactor { get; set; }
Property Value
Type Description
System.Double

The fudge factor.
Remarks

One should not be able to divide by 0, or raise 0 to a negative power. However, the simulator may not know this, so we introduce a fudge factor to remove this. If the exact solution is expected, use 0 as the fudge factor.
