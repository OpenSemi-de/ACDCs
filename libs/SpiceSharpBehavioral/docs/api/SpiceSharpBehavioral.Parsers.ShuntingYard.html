Class ShuntingYard

An expression parser that provides methods for parsing expressions using the Dijkstra's Shunting Yard algorithm. It can convert an expression in infix expression into Reverse Polish Notation (RPN).

Remarks

A value is only detected if a unary operator could not be found. So be careful when parsing for example "+1.5". If the leading "+" is not recognized as a unary operator, you may want to treat it as part of a value instead.
